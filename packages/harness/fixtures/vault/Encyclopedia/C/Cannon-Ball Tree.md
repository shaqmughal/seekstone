# Cannon-Ball Tree

CANNON-BALL TREE (_Couroupita guianensis_), a native of tropical South
America (French Guiana), which bears large spherical woody fruits,
containing numerous seeds, as in the allied genus _Bertholletia_ (Brazil
nut).
