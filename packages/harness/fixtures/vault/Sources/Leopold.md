# Leopold

LEOPOLD (M.H. Ger. _Liupolt_, O.H. Ger. _Liupald_, from _liut_, Mod.
Ger. _Leute_, "people," and _pald_, "bold," i.e. "bold for the people"),
the name which has been that of several European sovereigns.
